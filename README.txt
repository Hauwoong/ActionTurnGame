#  Action Turn Battle (가제)

Unity 기반 턴제 카드 전투 게임입니다.  
턴제 기반이지만 액션성과 연출을 강화하는 것을 목표로 합니다.

---

##  프로젝트 정보

- 엔진: Unity 2022 LTS
- 언어: C#
- 개발 인원: 1인 개발
- 개발 기간: 2026.01.23 ~

---

##  목표

- 턴제 카드 시스템 구현
- 카드 연출 강화
- 전투 흐름 구조화
- 유지보수 가능한 코드 설계
